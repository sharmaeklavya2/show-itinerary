# `show-itinerary`

This is a web app that takes a trip/journey in JSON and displays it as HTML.

## Setup

1.  Install typescript: `npm install -g typescript`.
2.  Build assets: `npm run build`.
3.  Serve this repository via a static HTTP server.

## How to run

1.  Open `index.html` in the web browser.
2.  Upload or drag-and-drop a JSON file.

## JSON format

See the `examples` directory to learn more about the JSON format.
